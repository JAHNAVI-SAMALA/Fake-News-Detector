# **Fake-News-Detector**
AI-powered Fake News Detection system that classifies news articles as Real, Fake, or Satirical using Machine Learning and NLP techniques. Built with Flask, Scikit-learn, and Bootstrap.

## **üöÄ Features**
- Detects Real vs Fake news articles
- Displays prediction confidence probability
- Clean and responsive Bootstrap UI
- Machine Learning-based text classification
- Flask web application deployment

## **üß† How It Works**
1. User inputs news article text
2. Text is cleaned and preprocessed
3. TF-IDF converts text into numerical features
4. Logistic Regression model predicts class
5. Result and confidence score are displayed
